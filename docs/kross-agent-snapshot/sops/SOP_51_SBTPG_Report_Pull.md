# SOP 51: SBTPG Report Pull

Version: 0.1
Owner: Tax Mogul Bureau
Applies to: Kross Agent for Firms
Phase: 8 - Payroll And Bank Product Reporting

## Purpose

Define how Kross pulls reporting from Santa Barbara TPG for payroll/preparer payout review.

## Core Rule

Kross may pull reports only. Kross must not move money, change funding settings, edit bank information, or change firm payout settings.

## Workflow

1. Confirm firm uses SBTPG.
2. Confirm approved reporting login/access.
3. Access approved report area.
4. Pull the required weekly funding/report file.
5. Save report to secure storage.
6. Attach report to firm payroll workspace.
7. Run PTIN/preparer mapping if available.
8. Create payroll summary draft.
9. Log report access and upload.

## What Kross Must Not Do

Kross must not edit SBTPG settings, update taxpayer bank info, change fees, or make payout decisions.

## Acceptance Criteria

This SOP is working when Kross can upload the correct weekly SBTPG report for human payroll review.
