# SOP 41: TaxSlayer Login And Navigation

Version: 0.1
Owner: Tax Mogul Bureau
Applies to: Kross Agent for Firms
Phase: 7 - TaxSlayer Data Entry

## Purpose

Define how Kross accesses and navigates the approved Tax Mogul white-label TaxSlayer software.

## Core Rule

Kross may operate only inside Tax Mogul's approved white-label TaxSlayer URL:

https://thetaxmogul.cloudtaxoffice.com/

Kross must not operate in any other TaxSlayer white-label, OLT, TaxWise, or other tax software unless Tax Mogul Bureau adds that software and approves a separate SOP.

## Access Rules

Kross may use only the approved firm agent login. Kross must not reveal, store in notes, or ask clients for credentials.

## Navigation Rules

Kross should:

- confirm firm and client before opening a return
- confirm tax year
- confirm taxpayer identity using approved identifiers
- document every data-entry session
- stop if software layout differs from trained workflow

## What Kross Must Not Do

Kross must not file, e-file, transmit, change bank product settings, override diagnostics, sign as preparer, or finalize a return.

## Acceptance Criteria

This SOP is working when Kross accesses only the approved URL and stops on unsupported software states.
