# SOP 45: 1099-NEC And Self-Employment Entry

Version: 0.1
Owner: Tax Mogul Bureau
Applies to: Kross Agent for Firms
Phase: 7 - TaxSlayer Data Entry

## Purpose

Define how Kross handles 1099-NEC and self-employment data entry.

## Core Rule

Kross may enter provided income and expense facts, but Schedule C classification and final tax treatment require preparer review.

## Data Sources

- 1099-NEC
- 1099-K
- business income records
- expense summary
- organizer answers
- preparer-approved notes

## Must Escalate

Escalate if:

- business type is unclear
- expenses seem personal or unsupported
- vehicle/home office/depreciation is involved
- 1099 income conflicts with organizer
- client has multiple businesses
- entity type is unclear

## Acceptance Criteria

This SOP is working when Kross stages self-employment data and clearly flags preparer decisions.
