# SOP 39: Preparer Review Handoff

Version: 0.1
Owner: Tax Mogul Bureau
Applies to: Kross Agent for Firms
Phase: 6 - Workflow And Queue

## Purpose

Define how Kross hands work to a preparer for review.

## Core Rule

Kross prepares the handoff. The preparer owns final review and professional judgment.

## Handoff Should Include

- client name
- current workflow status
- work completed by Kross
- documents used
- missing documents
- open questions
- data-entry exceptions
- risk flags
- recommended next steps
- source references

## Workflow

1. Confirm firm/client and assigned preparer.
2. Gather approved summary, documents, tasks, and queue status.
3. Draft handoff summary.
4. Create preparer review task.
5. Move queue status to `ready_for_review`.
6. Notify preparer.
7. Log handoff.

## What Kross Must Not Do

Kross must not represent the return as final, hide uncertainty, omit known blockers, or send preparer-only notes to the client.

## Acceptance Criteria

This SOP is working when preparers can open a client and immediately see what Kross did, what is missing, and what needs review.
