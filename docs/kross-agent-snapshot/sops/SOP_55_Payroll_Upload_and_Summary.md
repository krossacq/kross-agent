# SOP 55: Payroll Upload And Summary

Version: 0.1
Owner: Tax Mogul Bureau
Applies to: Kross Agent for Firms
Phase: 8 - Payroll And Bank Product Reporting

## Purpose

Define how Kross uploads payroll reports and creates payroll summaries inside the Preparer Workspace.

## Core Rule

Payroll summaries are drafts until approved by firm owner/admin.

## Workflow

1. Pull approved SBTPG or Refund Advantage report.
2. Upload file to secure payroll storage.
3. Attach to firm payroll area.
4. Map PTINs to preparers.
5. Draft payroll summary.
6. Flag unmatched records and exceptions.
7. Notify firm owner/admin for review.
8. Log upload and summary creation.

## Summary Should Include

- report date/range
- bank product platform
- number of funded returns
- preparers matched
- unmatched PTINs
- estimated payout by preparer
- exceptions
- source file reference

## What Kross Must Not Do

Kross must not issue payroll, move money, change payout settings, or expose payroll data to unauthorized users.

## Acceptance Criteria

This SOP is working when payroll reports and draft summaries are available for owner/admin review.
